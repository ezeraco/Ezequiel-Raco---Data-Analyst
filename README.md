# 📊 Telviso - Customer Subscription Analysis

An end-to-end Data Analytics project based on real customer subscription data from an Internet Service Provider (ISP). This project analyzes new customer registrations during June 2026 to identify sales trends, customer behavior, and business insights using Excel, SQL, Python, and Power BI.

> ⚠️ The dataset has been anonymized to protect confidential business information.

---

# 📖 Project Overview

The objective of this project is to simulate a real-world Data Analyst workflow, from data cleaning to dashboard creation.

The analysis focuses on answering business questions such as:

- Which subscription plans were the most popular?
- What was the expected monthly revenue?
- Which sales representatives generated the most subscriptions?
- How did subscriptions evolve throughout the month?
- What opportunities can be identified to improve business performance?

---

# 🎯 Business Objectives

- Analyze customer subscription behavior.
- Identify the most demanded products.
- Measure estimated monthly revenue.
- Evaluate sales performance.
- Build an interactive dashboard for decision making.
